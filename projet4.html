Projet 4 : Se présenter sur Internet

Présentation :

L'objectif de ce projet était de créer un site web sur le thème de notre choix.
Le site web possède une page d'accueil où l'on y retrouve les dernières actualités agricoles.
Celui-ci possède également une boutique redirigeant vers les boutiques officielles de John Deere, New Holland et Claas.
J'ai alors développer mes compétences en HTML et CSS.
Le site web est responsive et est validée par le W3C.

Avec ce projet, j'ai appris :

  • AC13.01 : Utiliser un système informatique et ses outils : acquis
  • AC13.04 : Connaître l’architecture et les technologies d’un site Web : acquis

Ma Contribution :

J'ai réalisé ce premier site web en autonomie, celui-ci respecte toutes les attendues de mon professeur. C'est-à-dire qu'il est adapté pour les téléphones(responsive), possèdes plusieurs pages et est validé par les normes W3C.
